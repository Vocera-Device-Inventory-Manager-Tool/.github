## 📌 About the Toolkit
**Vocera Device Inventory** is a Windows-based utility designed for organizations using Vocera communication hardware. It provides real-time inventory visibility, device assignment tracking, and usage metrics for large-scale deployments.

## ❓ What is Vocera Device Inventory?
This tool connects to the Vocera system database or management APIs and extracts comprehensive details about registered devices, including model, firmware version, assigned user, and last connection status. The results can be filtered, exported, or integrated into asset management systems.

## ✅ Why Choose Vocera Device Inventory?
- Centralized view of all Vocera devices  
- Simplifies tracking across departments or facilities  
- Quick identification of inactive or outdated devices  
- CSV and JSON export for audits and reporting  
- Secure API access using standard credentials  
- Ideal for IT teams in healthcare and enterprise environments  

## 🔧 Key Features
- Auto-discovery of Vocera devices on the network  
- Inventory sync with Vocera Voice Server or B3000/B4000 systems  
- Sorting and filtering by user, firmware, and location  
- Automated reporting and offline export options  
- Configurable refresh intervals for real-time updates  
- Role-based access control for audit compliance  
- Minimal resource footprint and fast operation  

---

## 🖥️ System Requirements
- Windows 10/11 (64-bit)  
- 4 GB RAM minimum  
- Network access to Vocera server environment  
- .NET Framework 4.8 or later  
- Administrative credentials for device queries  

## ▶️ Usage
1. Launch the utility and connect to your Vocera server or inventory API.  
2. Run a scan to populate device data.  
3. Filter by department, user, or firmware version.  
4. Export the report for auditing or recordkeeping.  
5. Schedule automatic refreshes if desired.
